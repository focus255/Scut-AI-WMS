## 附：数据库表关系

```
inbound_orders (入库单主表)
  ├── id (PK)
  ├── order_no (唯一单号)
  ├── status (未入库/已完成)
  └── supplier_code

inbound_details (入库明细)
  ├── id (PK)
  ├── inbound_id (FK → inbound_orders.id)
  ├── material_code
  ├── pack_capacity (箱容量)
  ├── plan_qty (计划数)
  └── actual_qty (实收数)

barcodes (条码追踪)
  ├── id (PK)
  ├── barcode (唯一条码号)
  ├── inbound_id (FK → inbound_orders.id)
  ├── material_code
  ├── supplier_code
  └── status (待入库/在库/已出库)

inventories (库存现存量)
  ├── material_code (唯一)
  ├── stock_qty (当前库存)
  ├── min_stock_days
  └── max_stock_days
```
